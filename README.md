# 半月预测 (Half-Month Stock Predictor)

基于历史股价数据的 **15 日涨跌幅预测** 工具。通过加权线性回归 + 动量分析，对中国 A 股主要企业的半月后价格走势进行预测。

## 功能特性

- 实时获取 Yahoo Finance 市场数据（支持 A 股 .SS .SZ 代码）（自动降级为模拟数据保障可用）
- 基于 60 个交易日的加权线性回归预测 + 近期动量调整
- 预测半个月（15 个自然日）后的股价与涨跌幅
- 可信度评分（综合 R^2、年化波动率、样本量）
- 交互式走势图表（Chart.js，90 天历史 + 15 天预测）
- 动态搜索添加任意美股代码
- 深色专业风格，响应式适配移动端

## 本地使用

项目为纯前端静态页面，无需构建工具：

```bash
cd outputs/stock-predictor
start index.html
```

> 注：本地 file:// 协议下 Yahoo Finance API 可能受 CORS 限制，
> 页面会自动降级为基于基准价格的模拟数据进行演示。
> **部署到 GitHub Pages 后可正常工作。**

## 部署到 GitHub Pages

1. 将 `outputs/stock-predictor/` 文件夹内容推送至 GitHub 仓库

2. 在仓库设置中启用 GitHub Pages：
   - 进入 Settings -> Pages
   - Source 选 **Deploy from a branch**
   - Branch 选 `main`，路径选 `/ (root)`

3. 部署成功后访问 `https://<用户名>.github.io/<仓库名>/`

## 预测方法

**数据源**: Yahoo Finance 近 90 个交易日历史收盘价

**回归模型**: 加权线性回归（指数衰减权重，近期数据权重更高）

**动量因子**: 最近 5 日均价 vs 前 10 日均价的偏差，调整幅度 30%

**可信度**: 基于 R^2 (0-40 分)、波动率 (0-30 分)、样本量 (0-20 分)、趋势稳定度 (0-10 分)

**预测周期**: 15 个自然日（约 11 个交易日）
